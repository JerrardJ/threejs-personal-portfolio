# ThreeJS Personal Portfolio
This project is a modern, interactive portfolio website built using React.js and styled with Tailwind CSS. The main highlight of this project is the integration of Three.js, enabling immersive 3D visuals and interactive elements that enhance user engagement and deliver a unique browsing experience.

The purpose of this project is to showcase personal skills, projects, achievements, and experience through a visually appealing and responsive interface. It also serves as a learning initiative to explore 3D rendering and WebGL concepts on the web.
